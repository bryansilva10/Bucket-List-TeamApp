#Bucket List App

Finally, a bucketlist you can store on the internet!

Uses Mongoose and Axios.

## For the Server:
### Scripts for Node
Run only server:
```
node run serve (will make database connection to mongoDB Atlas and also run localhost)
```
Run only client:
```
node run client
```
Run client and server concurrently:
```
node run dev
```
